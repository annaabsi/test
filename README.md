## Installation

### Dependencies

- nvm: [Node Version Manager](https://github.com/nvm-sh/nvm)
- yarn: [Package manager](https://yarnpkg.com/getting-started/install)

### Localhost

```bash
yarn install # install dependencies
yarn start # localhost:3000
yarn test # test
```
